# fastfetch-logo-picker

[English](#english) · [Dansk](#dansk)

A simple TUI/CLI logo viewer for **fastfetch**.

---

## English

### Requirements

- A POSIX-like shell environment
- `fastfetch` installed

### Install

1. Download/clone this repository.
2. Make the script executable:

```sh
chmod +x fastfetch-logo-picker
```

### Usage

Run:

```sh
./fastfetch-logo-picker
```

### Notes

- If you get a “permission denied” error, re-run the `chmod +x` step.
